![demo](./demo1.gif)

## Install

```
$ npm install --global alfred-transform-markdown
```

*Requires [Node.js](https://nodejs.org) 12+ and the Alfred [Powerpack](https://www.alfredapp.com/powerpack/).*

## Usage

In Alfred, type `trq`, <kbd>Enter</kbd>, and your query.
